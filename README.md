# Proyecto Reforestación — Predicción de Crecimiento de Plantas

Proyecto de Machine Learning del curso Inteligencia Artificial — Talento Tech (equipo Dataforest). El objetivo es predecir el crecimiento semanal de una planta (en cm) a partir de variables ambientales como temperatura, humedad relativa, horas de luz solar, pH del suelo, tipo de suelo, fertilidad, tipo de planta, categoría y departamento de Colombia.

Se entrenaron y compararon modelos de regresión (Regresión Lineal, Árbol de Decisión y Random Forest) sobre un dataset de 5000 registros sintéticos de especies como cacao, café, papa, maíz, yuca, entre otras. El target es la variable numérica Crecimiento_cm_semana.clear
